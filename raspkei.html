Raspkei Documentation

Raspkei is an automation application based in python for the Raspberry Pi.

Raspkei is currently under development and may not work as intended. Please keep this is mind when running.

Download

First, you'll need Git. 
git clone https://github.com/KeiBlackley/raspkei

Run

You'll need python3. 
python3 raspkei/app.py [arg]

Usage

Script

Arg Description
help Displays helpful information about the script and usable args.
update Runs an updater for the script and removes previous versions.
test Used by developers to test new features.
